# Personal Web
Bunch of tools for my personal usage.

## Technologies
Server
* Spring Boot

Client
* Typescript
* React
* Flux

## Start DEV mode
First version. Hopefully I will improve it in future.

* Run server
  * `mvn spring-boot:run`
* Run watchify
  * `cd src/main/resources/public`
  * `npm start`
* Use Atom to compile typescript.
